2ท อยุธยา แนะนำที่เที่ยว Offivial Website

1. วัดใหญ่ชัยมงคล

วัดใหญ่ชัยมงคล ถือได้ว่าเป็นอีกหนึ่งวัดสวย แลนด์มาร์คหนึ่งของจังหวัดอยุธยาเลยก็ว่าได้ สร้างขึ้นในสมัยอยุธยาตอนต้น คือในรัชสมัยของ สมเด็จพระรามาธิปดีที่ 1 หรือ พระเจ้าอู่ทอง จุดเด่นของวัดใหญ่ก็คือ พระเจดีย์ชัยมงคล ที่สร้างขึ้นในสมัยสมเด็จพระนเรศวรมหาราช ครั้งทรงมีชัยในยุทธหัตถี และด้านในได้มีการค้นพบชัยมงคลคาถาบรรจุอยู่ด้วย

2. ตลาดน้ำอโยธยา

ตลาดน้ำอโยธยา เป็นตลาดที่มีนักท่องเที่ยวคึกคักแทบทุกวัน สร้างขึ้นโดยมีคอนเซ็ปต์คงความเป็นกรุงเก่าของพระนครศรีอโยธยาในอดีตมาไว้ เพื่อให้เราได้อินกับบรรยากาศของกรุงเก่า แม้ไม่ใช่ตลาดเก่าดั้งเดิม แต่ก็มีกลิ่นอายของกรุงเก่าอย่างเต็มเปี่ยม เป็นอีกหนึ่งที่เที่ยวที่พลาดไม่ได้เลยถ้ามาเที่ยวอยุธยา

Learn more

3. วัดมหาธาตุ

วัดมหาธาตุ ตั้งอยู่ใน อุทยานประวัติศาสตร์พระนครศรีอยุธยา ที่นี่เป็นศูนย์กลางทางศาสนาที่สำคัญที่สุดในกรุงศรีอยุธยา เพราะเป็นวัดที่ประดิษฐาน พระบรมธาตุ ใจกลางพระนคร และเป็นที่พำนักของสมเด็จพระสังฆราชฝ่ายคามวาสีอีกด้วย สร้างในสมัย สมเด็จพระบรมราชาธิราชที่ 1 ขุนหลวงพะงั่ว เมื่อปี พ.ศ.1917 แต่แล้วเสร็จ และสร้างเพิ่มเติมจนเสร็จในสมัยสมเด็จพระราเมศวร

Learn more

4. ตลาดน้ำวัดท่าการ้อง

ตลาดน้ำวัดท่าการ้อง ตั้งอยู่ใน วัดท่าการ้อง วัดเก่าแก่ตั้งแต่สมัยกรุงศรีอยุธยา ที่นี่เป็นตลาดน้ำกรุงเก่า ที่เป็นแหล่งรวมของกินของฝากในราคาชาวบ้าน เราจะสามารถเดินชิลหาของอร่อยๆ รสชาติดั้งเดิมท้องถิ่น ราคาถูกแบบฝีมือคุณป้า คุณยายได้ที่นี่ได้เลย โดยเฉพาะก๋วยเต๋ยวเรืออยุธยาที่ต้องไปโดนสักที

Learn more

5. วัดนิเวศธรรมประวัติราชวรวิหาร

วัดนิเวศธรรมประวัติราชวรวิหาร วัดสไตล์โกธิก หนึ่งเดียวในประเทศไทย วัดตั้งอยู่บนเกาะลอยกลางแม่น้ำเจ้าพระยา โดยพระบาทสมเด็จพระจุลจอมเกล้าเจ้าอยู่หัว (รัชกาลที่ 5) โปรดฯ ให้สร้างขึ้น เพื่อทรงใช้เป็นสถานที่สำหรับบำเพ็ญพระราชกุศล เมื่อครั้งเสด็จฯ แปรพระราชฐานมาประทับที่พระราชวังบางปะอิน

Learn more

6. หมู่บ้านญี่ปุ่น อยุธยา

หมู่บ้านญี่ปุ่น อยุธยา เป็นอนุสรณ์สถานระหว่างไทย-ญี่ปุ่น โดยจะมีการนำเสนอประวัติศาสตร์ความเป็นมา วิถีชีวิต ความเป็นอยู่ อาชีพ ศาสนาของคนในหมู่บ้านญี่ปุ่นสมัยกรุงศรีอยุธยา

Learn more

7. วัดราชบูรณะ

วัดราชบูรณะ สร้างขึ้นในรัชสมัยของ สมเด็จพระบรมราชาธิราชที่ 2 หรือ เจ้าสามพระยา ในปี พ.ศ.1967 เป็นหนึ่งในวัดที่ใหญ่และมีความเก่าแก่มากที่สุดในอยุธยา ภายในวัดจะมี วิหารหลวง ทรงสี่เหลี่ยมก่อด้วยอิฐที่สวยงาม

Learn more

8. พิพิธภัณฑสถานแห่งชาติ เจ้าสามพระยา

พิพิธภัณฑสถานแห่งชาติเจ้าสามพระยา เป็นพิพิธภัณฑ์ที่จัดแสดงโบราณวัตถุต่างๆ ทั้งศิลปวัตถุจากการขุดค้นพบในพระนครศรีอยุธยา และยังมีโบราณวัตถุอีกมากมาย งานฝีมือช่างสมัยอยุธยา รวมถึงเครื่องไม้เครื่องมือของชาวบ้านในยุคกรุงเก่าเป็นอีกที่เที่ยวหนึ่งในอยุธยา ที่บอกเล่าเรื่องราวความรุ่งเรืองของยุคสมัยกรุงศรีอยุธยาได้เป็นอย่างดี

Learn more

9. ตลาดโก้งโค้ง

ตลาดโบราณ ในอยุธยากันที่ ตลาดโก้งโค้งที่นี่จะเปิดให้นักท่องเที่ยวได้มาเที่ยวช้อป ชิมกันเฉพาะทุกวันพฤหัสบดี-วันอาทิตย์ และวันหยุดนักขัตฤกษ์ โดยบรรยากาศภายในตลาดยังคงรักษาความดั้งเดิมของอยุธยาในสมัยก่อนๆ ไว้ มีลักษณะเป็นบ้านเรือนไทยหมู่ใหญ่ เป็นสถาปัตยกรรมไทยโบราณที่สวยงาม และมีข้าวของมากมายให้เราได้เลือกซื้อ ทั้งอาหารคาว อาหารหวานขนมโบราณ ผลไม้ต่างๆ จนไปถึงเครื่องไม้ เครื่องมือ อุปกรณ์เครื่องใช้ในบ้าน ของแต่งบ้านนานาชนิดทีเดียว

Learn more

10. ศรีอยุธยา ไลอ้อน ปาร์ค

ศรีอยุธยา ไลอ้อน ปาร์ค (Sriayuthaya Lion Park) ที่เที่ยวเปิดใหม่อยุธยา สวนสัตว์ บรรยากาศดี ที่ให้ทั้งครอบครัวได้ชิลสนุกสนาน ใกล้ชิดสัตว์นานาชนิดๆ บรรยากาศดี พื้นที่กว้างขวาง มีมุมถ่ายรูปทั่วไปหมดทั้งพื้นที่ งานนี้ เด็กๆ น้องๆ หนูๆ ต้องประทับใจมากๆ เลยทีเดียว

Learn more

11. วัดพนัญเชิงวรวิหาร

วัดพนัญเชิงวรวิหาร เป็นวัดเก่าแก่และสำคัญที่ต้องมาให้ได้สักครั้ง ที่นี่เป็นวัดที่มีประวัติมาอย่างยาวนาน มีการก่อสร้างมาตั้งแต่ก่อนการสถาปนากรุงศรีอยุธยา แต่ก็ไม่ปรากฎหลักฐานที่แน่ชัดว่าใครเป็นผู้สร้าง ภายในประดิษฐาน พระพุทธไตรรัตนนายก หรือ หลวงพ่อโต หรือ ซำปอกง พระพุทธรูปขนาดใหญ่ปางมารวิชัยที่นับว่าใหญ่ที่สุดในพระนครศรีอยุธยา และมีความศักดิ์สิทธิ์ เป็นที่เคารพบูชาของคนไทย

Learn more

12. ศาลหลักเมืองอยุธยา

เป็นสิ่งศักดิ์สิทธิ์ และมีความเก่าแก่ อยู่บ้านอยู่เมือง ตามพระราชพงศาวดารกรุงศรีอยุธยาหลายฉบับกล่าวว่า ศาลหลักเมือง มีมาตั้งแต่สมัยพระเจ้าอู่ทองนั้น ทรงสถาปนากรุงศรีอยุธยาขึ้นแต่ได้หักพังไปตามกาลเวลา และได้มีการสร้างศาลหลักเมืองขึ้นมาใหม่ ในสมัยสมโภชกรุงรัตนโกสินทร์ 200 ปี
